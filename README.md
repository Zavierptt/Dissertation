# Dissertation

## Background

This project was developed as part of a dissertation titled:

> *Beyond Black-Scholes: A Theoretical and Computational Investigation
> of Stochastic Volatility Models*

The dissertation examines the theoretical foundations and practical
calibration performance of Heston, Scott, and Hull-White models,
with a focus on their ability to reproduce market-observed
volatility smiles and skews.
